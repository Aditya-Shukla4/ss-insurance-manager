‚öôÔ∏è How to Run
==============

To run the SS Insurance Manager, follow these steps:

1. **Setup**: Clone the repository and navigate to the project directory.
2. **Environment**: Create a new file named `.env` and add the following environment variables:
	* `NEXT_PUBLIC_SUPABASE_URL`: The URL of the Supabase instance.
	* `NEXT_PUBLIC_SUPABASE_ANON_KEY`: The anonymous key of the Supabase instance.
	* `SUPABASE_SERVICE_ROLE_KEY`: The service role key of the Supabase instance.
3. **Build**: Run the command `npm run build` to build the application.
4. **Deploy**: Run the command `npm run start` to start the application.
